## 아이디어 요약

### 1. iMGuard
- 창구 출금 처리 시 인감·서명 유형(인감/서명/병행)을 자동 인식
- 최신 등록 원본 이미지를 자동 호출하여 전자문서와 AI 비교
- 일치율을 🟢 초록 / 🟡 주황 / 🔴 빨강 3단계로 표시, 불일치 시 전송 차단

### 2. WriteGuide
- 외국인 고객 신규 등록 시 입력된 국적 정보와 자동 연동
- 고객이 자국어로 입력 → 즉시 한글 변환 → 화면 보고 서류에 따라씀
- 자필 요건 충족 + 직원 설명 불필요 + 처리 시간 단축

### 3. ECHO
- 본부-영업점 전화 상담을 STT로 자동 텍스트화 및 요약
- AI가 Q&A 초안 자동 생성 → 담당자 검토 후 챗봇 즉시 등록
- 반복 전화 문의 감소 + 챗봇 지식 자동 업데이트 선순환 구축
