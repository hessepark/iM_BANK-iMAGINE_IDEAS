# WriteGuide — 외국인 고객 한글 작성 보조 시스템
> AI 기반 실시간 번역·한글 작성 보조 시스템 (Foreign Customer Korean Writing Assist System)

---

## 1. 프로젝트 개요

은행 창구에서 외국인 고객이 금융거래목적확인서 등 서류를 작성할 때,  
**한글로 직접 써야 하는 규정**을 지키면서도 고객과 직원 모두의 불편을 줄이는 시스템이다.

가장 간단한 해결책은 체크박스 선택 방식이나, 고객의 모국어로 입력받아 번역해서 보여주는 방식이다.  
그러나 금융 서류 특성상 **본인이 내용을 인지했다는 자필 증빙**이 반드시 필요하기 때문에,  
단순 선택이나 자동 입력 방식은 규정상 적용이 어렵다.

본 시스템은 이 규정을 준수하면서도,  
고객이 **자신의 언어로 입력 → 즉시 한글로 변환 → 한글을 보고 따라 씀** 이라는 흐름으로  
자필 요건을 충족하면서 작성 시간을 대폭 단축한다.

> "한글 작성 규정을 지키면서, 외국인 고객도 빠르고 정확하게 서류를 작성할 수 있는 창구 보조 시스템"

특히 외국인 고객 신규 등록 시 전산 화면에서 **국적이 필수 입력 항목**으로 지정되어 있어,  
이 시스템은 기존 전산 인프라와 **별도 추가 작업 없이 바로 연동**이 가능하다는 점에서 현실적 도입 가능성을 검토해볼 수 있다.

> 본 아이디어는 외국인 고객 내점 비율이 높은 **대학교 지점** 근무 경험을 바탕으로,  
> 실제 창구에서 반복적으로 겪은 불편함에서 출발하였다.

## 2. 문제인식
 
현재 은행 창구에서 외국인 고객이 서류를 작성할 때 다음과 같은 문제가 반복적으로 발생한다.
 
- 직원이 서류 항목을 **말로 또는 번역기로 설명**해야 하는 번거로움
- 고객이 내용을 이해하지 못하고 **자국 언어로 작성**하는 경우 발생
- 직원이 다시 **한글로 써달라고 요청** → 고객 당황 → 시간 지연
- 직원이 포스트잇에 한글로 번역해 보여줘도, **무슨 뜻인지 되묻거나 글씨를 알아보지 못하는** 경우 발생
- 한글을 모르는 고객이 **한 글자씩 따라 쓰는 데 시간이 오래 걸림**
- 직원은 옆에서 기다리며 **다른 업무 처리 불가**
 
특히 금융거래목적확인서, 주소, 직업, 학과 등  
**자유 기재 항목**에서 이 문제가 집중적으로 발생한다.
 
> 단순한 언어 장벽 문제를 넘어 → **창구 처리 시간 증가와 고객 불편**으로 이어지는 구조적 문제

## 3. 해결방안
 
**기존에 고려할 수 있는 방식과 한계**
 
| 방식 | 내용 | 한계 |
|------|------|------|
| 체크박스 선택 | 항목을 선택지로 제공 | 자유 기재 항목 대응 불가, 자필 증빙 미흡 |
| 자동 번역 입력 | 번역 결과를 시스템이 자동 기입 | 본인 자필 요건 미충족 |
| 현행 방식 | 직원이 직접 설명·요청 | 시간 과다 소요, 고객 불편 |
---

**본 시스템의 접근 방식**
 
외국인등록증 스캔 또는 국적 선택 시, 창구 보조 화면이 해당 언어로 전환된다.  
고객이 자국어로 항목을 입력하면 즉시 한글로 변환되어 크게 표시되고,  
고객은 그 한글을 보고 서류에 직접 따라 쓴다.
 
- 자필 요건 충족 (본인이 직접 한글로 작성)
- 직원의 반복 설명 불필요
- 한글을 몰라도 **모양을 보고 따라 쓰는 방식**으로 시간 단축
- 고객이 직접 입력한 내용이 전산 화면에 **바로 번역되어 표시**되므로 의미를 즉시 확인 가능
- 화면에 크게 표시된 글씨를 보고 따라 쓰므로 **필체 판독 문제 없이** 정확한 작성 가능

## 4. 주요기능

**국적 자동 감지 및 언어 전환**
- 외국인 고객 신규 등록 시 전산 화면에서 국적 입력이 필수 항목으로 지정되어 있어, 별도의 추가 입력 없이 기존 전산 데이터와 즉시 연동 가능
- 국적 정보를 기반으로 보조 화면 언어가 자동 전환되므로 직원의 별도 조작 불필요
- 현재 지원: 중국어, 영어 (확장 가능)

**서류 항목 다국어 안내**
- 금융거래목적확인서 등 작성 항목을 해당 언어로 안내
- 항목명과 작성 예시를 함께 표시

**실시간 번역 및 한글 변환**
- 고객이 자국어로 입력 시 즉시 한글로 변환
- 변환된 한글을 크게 표시하여 따라쓰기 유도

**직원 화면 자동 반영**
- 고객이 따라 쓰는 동안 직원 화면에도 동일 내용 순차 표시
- 별도 입력 없이 서류 작성 현황 확인 가능

**따라쓰기 안내**
- 변환된 한글 아래 자국어로 "따라써주세요" 안내 자동 표시

## 5. 업무 흐름

**현재 흐름**
```
외국인 고객 내방
→ 직원이 서류 설명 (말로, 번역기로)
→ 고객이 이해 못 하고 자기 언어로 씀
→ 직원이 다시 한글로 써달라고 요청
→ 고객 당황 + 시간 오래 걸림
```

**개선 흐름**
```
외국인 고객 내방
→ 외국인등록증 스캔 or 국적 선택
→ 창구 보조 화면에 해당 언어 안내 자동 표시
→ 고객이 보조 화면에 자기 나라 언어로 입력
→ 즉시 한글로 번역되어 크게 표시
→ 고객이 번역된 한글을 서류에 따라 씀
→ 자필 요건 충족 + 완료
```
